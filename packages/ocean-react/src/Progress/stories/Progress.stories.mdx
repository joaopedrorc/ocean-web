## CSS

| Global class               | Description                                     |
| -------------------------- | ----------------------------------------------- |
| .ods-progress              | Styles applied to the root element.             |
| .ods-progress--[sm,md,lg]  | Specifies the width of the element.             |
| .ods-progress\_\_primary   | Specifies the main circle style and its effect. |
| .ods-progress\_\_secundary | Specifies the style of the background circle    |

If that's not sufficient, you can check the [implementation of the component](https://github.com/ocean-ds/ocean-web/blob/master/packages/ocean-react/src/Progress/Progress.tsx) for more detail.
